SlimGems 1.3.9.2

A compatible and stable RubyGems1 fork

What is SlimGems?

SlimGems tries to bring some sanity to Ruby Package Management by giving the community a stable drop-in replacement of the RubyGems API, starting from the last stable version, 1.3.7. Install SlimGems and everything will work as it always did. The only difference: upgrading won't be scary anymore.

Project Goals

  1. A fast package manager that just works
  2. A consistent and stable API. No deprecations without ample2 warning to library developers
  3. Fostering a receptive and friendly community based around improving the package manager for everybody!

Get SlimGems

$ gem install slimgems
1 SlimGems is not RubyGems. It is a RubyGems fork. You can get RubyGems at rubygems.org. We're legally required to tell you this.
2 Deprecations would not mean warning messages or method removal. We will work with the community to make sure library developers upgrade their code on a convenient time frame for everybody (years, not months), and that nothing breaks. No rush!
