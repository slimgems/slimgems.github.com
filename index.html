SlimGems

A Compatible and Stable RubyGems✝ Fork

What is SlimGems?

SlimGems tries to bring some SANITY to Ruby Package Management by giving the community a stable drop-in replacement of the RubyGems API, starting from the last stable version, 1.3.7. Install SlimGems and everything will work as it always did. The only difference: upgrading won't be scary anymore.

Project Goals

  • A fast package Manager that Just Works™
  • A consistent and stable API. No deprecations without AMPLE warning to library developers. ✝✝
  • Fostering a receptive and friendly community based around improving the package manager for everybody!

Install SlimGems

$ gem install slimgems

Notes

✝ SlimGems is not RubyGems. It is a RubyGems fork. You can get RubyGems at http://rubygems.org. We're legally required to tell you this.

✝✝ Deprecations would not mean warning messages or method removal. We will work with the community to make sure library developers upgrade their code on a convenient time frame for everybody (years, not months), and that nothing breaks. No rush!
